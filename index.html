Comment Heberger et Partager avec tes Amis ?

3 methodes gratuites pour mettre en ligne tes cours :

Methode 1 : GitHub Pages (Recommande)

Etape 1 : Creer un compte sur github.com (gratuit)
Etape 2 : Creer un nouveau repository (ex: formation-ia)
Etape 3 : Uploader tous les fichiers (glisser-deposer le dossier rag_test)
Etape 4 : Aller dans Settings > Pages > Source: main > Save
Etape 5 : Ton site sera sur : https://tonpseudo.github.io/formation-ia/

Methode 2 : Netlify (Glisser-Deposer)

Etape 1 : Aller sur app.netlify.com/drop
Etape 2 : Glisser-deposer le dossier rag_test directement sur la page
Etape 3 : C'est en ligne ! Tu recois un lien instantanement

Methode 3 : Vercel

Etape 1 : Aller sur vercel.com, se connecter avec GitHub
Etape 2 : Importer le repository GitHub cree dans la methode 1
Etape 3 : Deploye automatiquement sur formation-ia.vercel.app
