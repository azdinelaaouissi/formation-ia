Phase 1 - Découverte 1 cours
1
ml_01_introduction.html - C'est quoi le Machine Learning ?
Définition, analogies du quotidien, IA vs ML vs DL, les 3 types d'apprentissage (supervisé, non supervisé, renforcement), histoire (1950→2026), applications réelles (Netflix, Spotify, médecine, voitures), les métiers du ML, mythes vs réalité, workflow ML global en schéma
~15 chapitres ~2h Débutant
Phase 2 - Fondations 2 cours
2
ml_02_maths_visuelles.html - Les Maths du ML (version visuelle)
Vecteurs (flèches), matrices (tableaux), statistiques (moyenne, médiane, écart-type avec histogrammes), probabilités (dés, cartes), distributions (courbe de Gauss), fonctions (pente = dérivée), gradient descent (la bille qui roule), coût/erreur. Tout en schémas, zéro formule complexe
~12 chapitres ~2h30 Débutant
3
ml_03_donnees.html - Comprendre les Données
Types de données (numériques, catégorielles, texte, images), features et labels, datasets célèbres (Iris, MNIST, Titanic), train/test/validation split, preprocessing (normalisation, standardisation, encodage), feature engineering, données manquantes, biais dans les données, qualité des données, data augmentation
~14 chapitres ~2h Débutant
Phase 3 - Apprentissage Supervisé 3 cours
4
ml_04_regression.html - La Régression
Régression linéaire simple (la droite), régression linéaire multiple, régression polynomiale (la courbe), régularisation (Ridge, Lasso, ElasticNet), analogies (prédire le prix d'une maison), métriques (MSE, RMSE, MAE, R²), overfitting vs underfitting en régression, schémas de toutes les courbes
~12 chapitres ~2h30 Intermédiaire
5
ml_05_classification.html - La Classification
Régression logistique (seuil de décision), KNN (les voisins), SVM (la frontière), arbres de décision (l'arbre de questions), Naive Bayes (les probabilités), métriques (accuracy, precision, recall, F1), matrice de confusion, courbe ROC/AUC, multi-classe, schéma de chaque algo
~14 chapitres ~3h Intermédiaire
6
ml_06_ensemble.html - Méthodes d'Ensemble
Sagesse des foules, bagging vs boosting (schéma), Random Forest (la forêt), Gradient Boosting (les correcteurs), XGBoost, LightGBM, CatBoost, stacking (l'équipe), voting, comparaison de tous les ensembles, quand utiliser quoi, avantages/inconvénients
~12 chapitres ~2h Intermédiaire
Phase 4 - Apprentissage Non Supervisé 1 cours
7
ml_07_non_supervise.html - Apprentissage Non Supervisé
Clustering (K-Means pas à pas, DBSCAN, Hierarchical), comment choisir K (méthode du coude), réduction de dimension (PCA en 2D, t-SNE), détection d'anomalies, règles d'association (panier de courses), segmentation clients, comparaison des algorithmes de clustering
~12 chapitres ~2h30 Intermédiaire
Phase 5 - Réseaux de Neurones et Deep Learning 3 cours
8
ml_08_reseaux_neurones.html - Réseaux de Neurones
Le neurone biologique vs artificiel, perceptron, fonctions d'activation (sigmoid, ReLU, tanh, softmax) avec graphiques, couches (input, hidden, output), forward propagation pas à pas, backpropagation (l'erreur qui remonte), gradient descent visuel, learning rate, epochs, batch size
~14 chapitres ~3h Intermédiaire+
9
ml_09_deep_learning.html - Deep Learning
CNN (filtres sur images, convolution, pooling, architectures célèbres), RNN (séquences, mémoire courte), LSTM (mémoire longue, portes), GRU, Transformers (attention is all you need), GANs (le faussaire vs le policier), autoencoders, transfer learning, GPU vs CPU
~16 chapitres ~3h30 Avancé
10
ml_10_nlp.html - NLP (Traitement du Langage)
Comment un ordinateur lit du texte, tokenization (mots, sous-mots, BPE), Bag of Words, TF-IDF, Word2Vec (roi - homme + femme = reine), GloVe, Sentence Transformers, BERT, GPT, attention mechanism visuel, du texte au RAG (le pont avec tes cours précédents)
~14 chapitres ~2h30 Avancé
Phase 6 - Maîtrise 2 cours
11
ml_11_evaluation.html - Évaluation et Optimisation
Overfitting vs underfitting (schéma clair), biais vs variance, cross-validation (k-fold), confusion matrix détaillée, toutes les métriques (accuracy, precision, recall, F1, AUC), hyperparameter tuning (Grid Search, Random Search, Bayesian), early stopping, learning curves
~12 chapitres ~2h30 Intermédiaire+
12
ml_12_pipeline.html - Pipeline ML Complet
Workflow complet de A à Z : problème → données → exploration → preprocessing → feature engineering → sélection modèle → entraînement → évaluation → déploiement. MLOps basics, versioning modèles, monitoring en production, A/B testing, schéma pipeline complet
~12 chapitres ~2h Avancé
